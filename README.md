# precious car

## サイト概要

### サイトテーマ.

- 自分の愛車の投稿、他のユーザーの愛車の投稿閲覧
- 投稿車種の検索ができるコミュニティサイト

### テーマを選んだ理由

幼い頃から車が好きで2022年に念願の新車のスポーツカーを納車しました。
他の人がどんな「きっかけ」、「こだわり」を持っているか興味があり
お互いに交流ができると便利だと思いこのテーマにしました。

### ターゲットユーザー

- 愛車を自慢したい人
- 車を大切にしている人
- 車好き同士で交流したい人

### 主な利用シーン

- 自分の車を投稿する時
- 他の人の車を見たい時
- 興味のある車種を検索する時
- コメントを介して交流したい時
